# Woah Games

A React Native mobile gaming app built with Expo.

## Games
- Tic-Tac-Toe (two-player local)
- Rock Paper Scissors (vs. computer, with persistent stats)
- Higher or Lower (card guessing game, with persistent best score)

## Tech Stack
- React Native + Expo Go
- React Navigation (stack navigator)
- AsyncStorage (persistent score tracking)
